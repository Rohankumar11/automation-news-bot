# Setup Guide — PGDM Daily News Bot

## Prerequisites
- n8n Cloud account (mutashi.app.n8n.cloud or any instance)
- Google AI Studio account (for Gemini API)
- Telegram account
- Basic understanding of n8n workflows

---

## Step 1 — Telegram Bot Setup
1. Open Telegram → search **@BotFather**
2. Send `/newbot` → follow prompts → get your **Bot Token**
3. Send any message to your new bot
4. Visit `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`
5. Copy the `"id"` value inside `"chat"` → this is your **Chat ID**
6. Save both values in your `.env` file

---

## Step 2 — Gemini API Setup
1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Click **Get API Key** → Create new key
3. Copy the key → save in `.env` as `GEMINI_API_KEY`
4. Recommended model: `gemini-1.5-flash-8b` (free tier friendly)

> ⚠️ Do NOT commit your API key to GitHub. Use `.env` only.

---

## Step 3 — Import Workflow into n8n
1. Log in to your n8n instance
2. Click **+ New Workflow**
3. Click the **3-dot menu** (top right) → **Import from file**
4. Select `workflow/pgdm_news_bot.json`
5. Workflow loads with all nodes intact

---

## Step 4 — Add Credentials in n8n

### Gemini
1. Click the **Google Gemini Chat Model** node
2. Click **Credentials** → **Create New**
3. Paste your `GEMINI_API_KEY`
4. Save

### Telegram
1. Click the **Send a text message** node
2. Click **Credentials** → **Create New**
3. Paste your `TELEGRAM_BOT_TOKEN`
4. Save

---

## Step 5 — Configure RSS Sources
The workflow uses 3 RSS Feed nodes. Default sources:

| Node | Source | URL |
|------|--------|-----|
| RSS Read | Reuters Business | `https://feeds.reuters.com/reuters/businessNews` |
| RSS Read1 | Moneycontrol | `https://www.moneycontrol.com/rss/business.xml` |
| RSS Read2 | TechCrunch | `https://techcrunch.com/feed/` |

To change a source, click the RSS node → update the **Feed URL** field.
